--Proyecto Final: Figuras

Este proyecto es una aplicacion en  que puede crear diferentes figuras y calcular su área y perimetro
El programa utiliza **Programación Orientada a Objetos**, **herencia**, **polimorfismo**, **clases abstractas** y **manejo de excepciones**.
El programa utiliza recursos como: herencia, polimorfismo y maneja  excepciones.


--Estructura de clases

Figura.java
Circulo.java
Rectangulo.java
Triangulo.java
MainFigura.java

--Como EJECUTAR el programa
Se incia y sale el menu con las 5 opciones
     - 3 (1-3) sirven para crear figuras, cuando las elijas introduces los datos para 
      necesarios para calcular sus caracteristicas, segun esos datos
     - 4 sirve para mostrar un resumen de las figuras que has creado anteriormente, si no has creado niguna sale un mensaje
     - 5 sirve para cerrar el programa ya el programa no se cierra despues de cada ejecucion
       porque si no no podrias crear mas de 1 figura y el resumen no tendria sentido
